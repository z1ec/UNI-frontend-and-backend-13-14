# Заметки PWA

Учебный проект для практических заданий 13-18 по фронтенд- и бэкенд-разработке. Приложение реализует заметки в формате PWA, кэширование App Shell, работу офлайн, обмен событиями через WebSocket, push-уведомления и напоминания с возможностью отложить уведомление.

## Возможности

- Добавление обычных заметок.
- Добавление заметок с датой и временем напоминания.
- Хранение заметок в `localStorage` со структурой `id`, `text`, `reminder`, `timestamp`.
- Регистрация Service Worker и кэширование App Shell для офлайн-режима.
- Подписка и отписка от push-уведомлений через VAPID.
- Планирование напоминаний на сервере через `setTimeout` и хранение активных таймеров в `Map`.
- Push-уведомление для напоминания с действием `Отложить на 5 минут`.
- Обработка действия уведомления в Service Worker через POST-запрос `/snooze`.

## Структура

- `index.html` - основная страница приложения.
- `content/home.html` - формы добавления заметок и список записей.
- `content/about.html` - описание приложения.
- `app.js` - клиентская логика, `localStorage`, WebSocket и подписка на push.
- `sw.js` - Service Worker, кэширование, показ push-уведомлений и обработка действий.
- `server.js` - Express/HTTPS-сервер, Socket.IO, подписки push, планирование и перенос напоминаний.
- `manifest.json` - манифест PWA.

## Запуск

Установите зависимости:

```bash
npm install
```

Запустите сервер:

```bash
npm start
```

Откройте приложение:

```text
https://localhost:3001
```

Если локальные сертификаты `localhost.pem` и `localhost-key.pem` недоступны, сервер автоматически запустится по HTTP. Для Service Worker и Push API в браузере рекомендуется использовать HTTPS или `localhost`.

## Проверка

1. Откройте приложение и убедитесь, что Service Worker зарегистрирован.
2. Нажмите `Включить уведомления` и разрешите уведомления в браузере.
3. Добавьте обычную заметку и проверьте появление записи в списке.
4. Добавьте напоминание с датой в будущем.
5. Дождитесь push-уведомления.
6. В уведомлении нажмите `Отложить на 5 минут` и убедитесь, что сервер переносит напоминание.
